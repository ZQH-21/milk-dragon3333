# BUPT-TA-Recruitment-Group33

This repository contains the group coursework for EBU6304 Software Engineering Group Project. The project aims to develop a Teaching Assistant Recruitment System for BUPT International School using Agile methodologies, including product backlog, prototype, core features, and supporting documentation.

This project is a JSP/Servlet-based prototype for a Teaching Assistant Recruitment System. The repository is organized so that documentation, source code, view pages, and static assets are easier to locate and maintain.

## Project Structure

```text
.
|-- docs/                    Project documents and coursework artifacts
|-- webapp/
|   |-- images/              Static assets
|   |-- WEB-INF/
|   |   |-- src/
|   |   |   |-- controller/  Servlet controllers
|   |   |   |-- model/       Domain models
|   |   |   `-- store/       File-based persistence helpers
|   |   |-- views/
|   |   |   |-- mo/          MO-facing JSP pages
|   |   |   `-- ta/          TA-facing JSP pages
|   |   `-- web.xml          Servlet mappings
|   |-- start.html           Login and registration entry page
|   `-- *.jsp                Compatibility entry pages forwarding to internal views
|-- request.txt              Task notes
`-- README.md
```

## Notes

- External access paths are kept unchanged for compatibility.
- JSP pages are grouped by role under `webapp/WEB-INF/views`.
- Java sources are organized by responsibility under `controller`, `model`, and `store`.
